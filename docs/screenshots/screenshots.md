# Screenshots

This folder contains screenshots that serve as visual proof of the **Azure Enterprise Landing Zone** implementation. The images document key configurations and resources deployed in the Azure environment and support validation of the project setup.

## Purpose

The screenshots provide visual confirmation that required Azure components, governance controls, and configurations were successfully implemented as part of the landing zone architecture.

## Contents

Screenshots in this folder may include:

* **Management Group hierarchy**
* **Subscription structure**
* **Resource group organization**
* **Policy definitions and assignments**
* **Role-Based Access Control (RBAC) configuration**
* **Virtual network and subnet configuration**
* **Security configurations (Defender for Cloud, policies, etc.)**
* **Deployed platform or workload resources**

## Notes

* Screenshots should clearly display the relevant Azure Portal configuration.
* Sensitive information (such as subscription IDs, tenant IDs, or email addresses) should be hidden or blurred if necessary.
* Each screenshot should correspond to a specific configuration or requirement within the project.
